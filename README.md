# Hotel Praia — Frontend 🏨

Aplicación web para la gestión integral de un hotel: consulta de habitaciones, sistema de reservas, autenticación de usuarios, panel administrativo, gestión de clientes, pagos y reseñas.

🔗 **Demo en vivo:** [hotel-praia-front.vercel.app](https://hotel-praia-front.vercel.app/)
🔗 **Repositorio del backend:** [hotel-praia-back](https://github.com/thomaslopezro/hotel-praia-back)

---

## 🛠️ Stack Tecnológico

### Framework principal
- **Angular 16.2** + **TypeScript 5.1**
- **RxJS 7.8** para programación reactiva
- **Angular CLI 16.2.16**
- **SCSS** para estilos

### UI y estilos
- **Tailwind CSS** (vía CDN)
- **ngx-bootstrap 11.0.2** — componentes Bootstrap para Angular
- **Bootstrap Icons 1.13.1**

### Testing
- Karma 6.4 + Jasmine 4.6

### Deployment
- **Vercel** (CI/CD automático)

---

## 📋 Funcionalidades

- 🏠 Visualización de habitaciones disponibles
- 📅 Sistema de reservas con validación de fechas
- 🔐 Login / registro de usuarios con autenticación JWT
- 👨‍💼 Panel de administración para empleados
- 👥 Gestión de clientes
- 💳 Sistema de pagos
- ⭐ Calificaciones y reseñas
- 📱 Diseño responsive

---

## 🚀 Cómo correr el proyecto

### Prerrequisitos
- Node.js 18+
- Angular CLI 16.x
- Backend corriendo (ver [hotel-praia-back](https://github.com/thomaslopezro/hotel-praia-back))

### Instalación

```bash
git clone https://github.com/thomaslopezro/hotel-praia-front.git
cd hotel-praia-front
npm install
ng serve
```

La aplicación estará disponible en `http://localhost:4200`
